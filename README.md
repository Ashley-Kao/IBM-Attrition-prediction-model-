# IBM-Attrition-prediction-model-

The project aims to predict employees likely to leave and analyzes the IBM HR Attrition dataset to identify factors contributing to employee turnover. 
It uses the dataset to shows the machine learning pipeline from data loading to insights.

- **Dataset:** IBM HR Attrition Dataset  
- **Source:** [Kaggle - IBM Attrition Dataset](https://www.kaggle.com/datasets/yasserh/ibm-attrition-dataset)  
- **Size:** 1,470 employee records with 35 features

Tools used

- Python (Jupyter Notebook)
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
